Getting started
===============

.. admonition:: What is Hasura GraphQL engine?

  Watch this `4 minute video <https://www.youtube.com/watch?time_continue=2&v=oFRwAFWXHuE>`_ to know more. If you
  have any queries kindly ping us on chat.

To get started, you need to run the Hasura GraphQL engine and set up the console by installing the Hasura CLI. You
can also choose to run on top of your existing Postgres database.
